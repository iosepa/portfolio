The process.

  1. We’ll discuss if we are a good fit.
    • Our discussion will include scope, timelines, how to determine if the website is a success. Links to website or two that you admire are useful for a starting point.
    • If we're not a good fit I can refer you to another web design company.
  2. We match (yay!) we’ll discuss timelines (for development/design and content), expectations, and sign a contract.
  3. I’ll build a mockup and get your approval for the serious work.
    • If we decide on wordpress and you’d like to manage the site in-house I’ll provide training on caring for the site.
    • We’ll go over SEO and how to ensure your website stays speedy.
    • Sometimes the longest part of the process can be waiting for copy.
  4. You'll get weekly progress updates.
  5. We’ll do a dry-run, make sure everything works and bring it live.
  6. We can discuss if you want me to be involved with updating/hosting the site.

Why wordpress?

Wordpress is a content management system (CMS). The designer/developer codes up how the site will look (the theme) and behave (plug-ins). Wordpress has an admin portal which allows the site owner to login and update content, add pages and even update wordpress-all without needing to contact the developer or mess with code.

Wordpress advantages:
  • Powers 30% of the web circa Jan 2019.
  • Security updates
  • Many useful plugins which can help with SEO, content management, forms etc…
  • Large developer base, many developers are familiar and can help with existing sites.
Disadvantages:
  • The plugins and options come at the cost of speed.
  • CMS is developed and updated by a third party.
